# Git Branch

### 1st Time:

- git pull origin main
- git checkout -b <branch_name>
- git add . ; git commit -m “...”
- git push origin <branch_name>


### Other Time:

- git checkout main
- git pull origin main
- git checkout <branch_name>
- git add . ; git commit -m “...”
- git push origin <branch_name>
